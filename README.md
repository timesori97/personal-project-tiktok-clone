# personal-project-tiktok-clone

바이코딩으로 7주일 걸려서 만든 프로젝트입니다

핵심 기능 요약
🎥 영상 서비스


영상 업로드 및 관리: 사용자가 비디오 파일과 제목, 설명을 입력하여 서버에 업로드할 수 있으며, 업로드 시 UUID를 통해 파일명 중복을 방지합니다.


메인 피드 (인피니트 스크롤 스타일): Intersection Observer를 사용하여 사용자가 보고 있는 영상이 화면의 70% 이상 노출될 때 자동 재생되는 틱톡 특유의 UX를 구현했습니다.


검색: 영상의 제목이나 설명 키워드를 통해 관련 영상을 찾을 수 있습니다.
🤝 소셜 인터랙션


좋아요 및 북마크: 영상에 대해 좋아요를 누르거나 나중에 볼 수 있도록 저장(북마크)할 수 있으며, 토글 방식으로 취소 기능도 포함되어 있습니다.


실시간 댓글: CommentsModal을 통해 영상별로 댓글을 작성하고 목록을 실시간으로 확인할 수 있습니다.


팔로우 시스템: 다른 유저를 팔로우하거나 팔로워/팔로잉 숫자를 확인할 수 있는 기능을 갖추고 있습니다.
👤 사용자 관리


회원가입 및 보안 로그인: BCryptPasswordEncoder를 사용하여 비밀번호를 암호화하여 저장하며, JWT(JSON Web Token) 발급을 위한 구조가 준비되어 있습니다.


프로필 편집: 자신의 프로필 이미지 업로드, 자기소개(Bio) 수정, 비밀번호 변경이 가능합니다.
🛡️ 관리자(Admin) 모드


대시보드: 총 가입 유저 수, 누적 영상 수, 미처리 신고 건수를 한눈에 파악할 수 있는 통계 기능을 제공합니다.

신고 처리: 유저가 부적절한 영상을 신고하면 관리자가 이를 확인하고 해당 영상을 즉시 삭제(Soft Delete) 처리할 수 있는 중재 시스템이 구축되어 있습니다.
